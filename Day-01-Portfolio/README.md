# 🌐 Day 01 - Portfolio Website

## 🚀 Project Overview

This is my **Day 01 project** of the *100 Days of Frontend Development Challenge*.
I built a clean and responsive **Portfolio Website** using HTML and CSS.

The goal of this project was to understand:

* Basic HTML structure
* CSS styling and layout
* Responsive design principles
* Simple hover interactions

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## ✨ Features

* Responsive layout
* Modern UI design
* Smooth scrolling navigation
* Project cards with hover effects
* Contact form UI

---

## 🎯 What I Learned

* Structuring a webpage using semantic HTML
* Styling using CSS variables and flexbox/grid
* Creating hover effects and transitions
* Improving UI/UX with spacing and typography
